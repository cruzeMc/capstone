PRODUCT OVERVIEW

In light of all these challenges faced by the promoters and patrons we have decided to undertake the DreamVent project. DreamVent, as we envision, will be a product that brings together promoters and patrons in a way enhances the experiences and rewards they get from events. The web application will be a platform where promoters can not only post their event but will be able to make the necessary planning arrangements using the app as well. These arrangements include accessing statistical data based on reviews, visits and other data gathered from the patrons. For the patrons, the web app will allow them to access an organized database of events so that they will be able to continually follow the trends and changes of a particular event(s). If they are registered, events will be filtered to them based on their preferences, personality, geographic location and browsing habits within the application. The patron will also be able to purchase tickets to the event via the web application and conduct refunds and other monetary transactions on this convenient online platform. This web application will systematically cover most of the bases on the parts of both the promoter and patrons in a way that the other promotional methods could not. In the end, this app will result in a cheaper, more technologically advanced and creative way to market and bring to the fore one of the most dynamic mainstays of the Jamaican economy.

WHAT IT OFFERS TO USERS

1. The system shall allow users to create profiles in the categories of promoter or patron.

2. The system shall allow the collection of data live data from patrons,base on their interaction with the system.

3. The system shall analyze data on users and users’ behaviour to form the basis of a recommendation system

4. The system shall recommend events to patrons based on personal preferences.

5. The system shall rely on mathematical and computational techniques to recommend events.

6. The system shall allow patrons to rate events

7. The system shall allow patrons to comment on events

8. The system shall allow users to view location of the events on Google Map

9. The system shall allow the promoters to view statistics about their events

10. The system shall allow the promoter to do a whatif analysis

11. The system shall takes into account the variables that affect the patrons, then allow the promoter to manipulate them, to see its    effect - all of this shall be done in a sandbox environment

12. The system shall allow the analysis of comments to determine whether it is negative or positive

13. The system shall use this analyzed comment data to provide feedback to promoters and also recommend events to patrons

14. The system shall allow patrons to purchase tickets

15. The system shall provide notifications to users about events

System Manual

  • Landing

    When the software is opened the user will first encountered the landing page with the links to each category of events.

  • How to login or sign up

    At the top of the landing page is dropdown menu with a link to either signup or login

  • How to login

    If the user wishes to login he/she will be redirected to the login form after which the user should fill in his/her credentials and submit

  • How to sign up

    If the user wishes to sign up he/she will be redirected to the signup form after which he she is expected to fill out the form

  • How to get the create event link

    To create an event the user is required to be logged in and from there he/she can select the Create event link

  • How to create event

    To create the event complete the event form

    Created events will appear within the category that they match.

  • How to get event details

    To get the details about an event click on the event and it will redirect to the page with the event details.

  • How to add comments

    To comment on events users place the comment on the input field

  • How to rate events

    To rate events users select the icon on the bottom right of the event details page and select the corresponding rating. Ratings are on a scale of 1-5.

  • How to Purchase tickets

    To purchase tickets for an event select the purchase link on the corresponding event.

  • How to view recommendations

    Logged in users can see their recommended events by selecting the home link on the menu

    The recommended events appears on a page with the time tracker for the number of days until the event

  • How to Update Profile

    To update profile information select the Account link on the dropdown menu and it will redirect you to the form to add the updated information.

  • How to logout

    To logout select the logout link from the dropdown menu.
